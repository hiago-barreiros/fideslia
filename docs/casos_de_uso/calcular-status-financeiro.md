# Calcular Status Financeiro da Proposta

### Ator
Sistema

### Descrição
Calcula o status financeiro de uma proposta com base
nos pagamentos confirmados.

### Regras de negócio
- Total pago é a soma dos pagamentos confirmados
- Saldo é um valor derivado
- Status financeiro não é persistido

### Estados financeiros
- Aberta
- Em andamento
- Quitada
