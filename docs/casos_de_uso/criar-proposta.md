# Criar Proposta

## Ator
Prestador

## Descrição
Permite que o prestador crie uma proposta associada a um cliente.

## Pré-condições
- Cliente existente

## Fluxo principal
1. Prestador informa título e descrição
2. Proposta é criada com status "rascunho"

## Regras de negócio
- Proposta inicia sempre em rascunho

## Pós-condições
- Proposta disponível para edição ou envio
